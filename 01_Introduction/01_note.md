# Day 1 - 入門概論

## Environment Setup

檢查目前安裝的 Python 版本  

```py
python --version
```

使用指令開啟 Python Interactive shell

```console
python
```

關閉並離開 Python Interactive Shell，也可以使用 `ctrl + d`

```python
exit()
```

如果執行的內容有錯誤，Python 會顯示相關的錯誤 (Error) 訊息內容，因此可以多瞭解常見的 Python 錯誤訊息類型。

常見的 `error types`: `SyntaxError`, `IndexError`
